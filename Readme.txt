"Pomodoro"

Представляю вам приложение "Pomodoro" - инструмент для повышения продуктивности, основанный 
на знаменитой технике Pomodoro.

Основные возможности:

Таймер Pomodoro:
Установите продолжительность интервалов работы и отдыха по своему вкусу.
Начните таймер и наблюдайте за его работой, приложение уведомит вас о начале и конце 
каждого интервала.
Продолжайте работать, пока не закончите текущую сессию, или приостановите таймер в любой 
момент.

Статистика:
Отслеживайте свои успехи! Приложение записывает продолжительность каждой сессии работы и 
отдыха.
Графически визуализируйте свои достижения за день, неделю, месяц - отслеживайте прогресс и 
анализируйте свою продуктивность.

Сохранение данных:
Все настройки и статистика сохраняются автоматически, вам не нужно беспокоиться о потере 
прогресса.
Ваши данные доступны в любое время, независимо от устройства.

Демонстрация:
Установка таймера:
На странице "Таймер" вы увидите два поля для ввода: "Продолжительность работы" и 
"Продолжительность отдыха".
Задайте желаемое время для каждого интервала, например, 25 минут для работы и 5 минут для 
отдыха.
Нажмите кнопку "Старт" для запуска таймера.

Работа с таймером:
Во время сессии работы таймер отсчитывает время, информируя вас о оставшемся времени до 
конца интервала.
По завершении интервала работы приложение издает уведомление и переходит к интервалу 
отдыха.
Вы можете остановить таймер в любое время, нажав кнопку "Стоп", и возобновить его, нажав 
"Старт".

Просмотр статистики:
Перейдите на страницу "Статистика".
Выберите период, за который вы хотите просмотреть данные (день, неделя, месяц).
Просмотрите график, показывающий продолжительность каждой сессии работы и отдыха за 
выбранный период.
Анализируйте свои результаты и делайте выводы о своей продуктивности!

Преимущества:
Улучшенная концентрация: Помогает сосредоточиться на работе, разбивая время на короткие 
продуктивные сессии.
Повышенная продуктивность: Позволяет максимально эффективно использовать время, избегая 
переутомления.
Гибкость: Настраиваемые интервалы позволяют адаптировать приложение к индивидуальным 
потребностям.
Простой интерфейс: Легко использовать, приложение интуитивно понятно.
Как запустить приложение:
Установка зависимостей:
npm install

Запуск JSON-сервера (для разработки):
npm run api

Запуск приложения:
npm run dev

Приложение откроется в браузере по адресу http://localhost:5173/.
